Move to AD and move between the OUs
-----------------------------------
# Moving to Active Directory
# Creating an Organizational Unit
# Move Active Directory users from one OU to another OU


New-ADOrganizationalUnit -Name "Test-User" : Creating an OU.
Get-ADUser -Identity "user"

Please refer to the screenshot 1 to see full commands. You will see after moving the user that the Distinguished name will change.

Using a pipeline now moving the user back to the original OU:

Get-ADUser -Identity "user" | Move-ADObject -TargetPath "OU=Test,DC=Local,DC=qhamaninanderebe,DC=Com"

And the get the user to verify that.
